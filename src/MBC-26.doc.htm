 Description   

Описание:
Нужно добавить по умолчанию к каждой рассылке возможность отписки (в настройках рассылки http://crm.mixtraff.com/?r=domain/emailPrepare&num=10 добавить галку отключения этой функции, чтобы не показывалась надпись ). Текст размещаем в самом конце письма. Таблица с теми кто отписался и возможность добавлять новые почтовые ящики. После перехода по ссылке отписки, человек попадает на страницу с опросником где он указывает причину из нескольких вариантов по умолчанию забитых в систему и одного текстового поля куда можно написать свою причину.

Текст отписки:
Если Вы не хотите получать рассылку, перейдите по ссылке: отписаться

Интерфейс:
1) На странице http://crm.mixtraff.com/?r=domain/emailPrepare&num=10
Добавляем чекбокс https://monosnap.com/file/4KBHKHtfr7kHeIVAWKmUcjkrxv6kJo
По умолчанию он всегда включен ( при отправке писем есть надпись с отпиской ).
2) Отдельная страница с теми кто перешел по ссылке и отказался от рассылки в виде таблицы.
Размещаем ее в меню "контакт".

2.1 Таблица:

Площадка Email Причина Трафик Месяц Страны
Столбец A1 Столбец A2 Столбец A3 Столбец A4 Столбец A5

Трафик Месяц = SimilarWeb статистика
Страны = SimilarWeb статистика

2.2 Возможность с этой страницы добавлять почтовые ящики в черный список самостоятельно,
после добавления почтового ящика в таблице по этому почтовому ящику должен показываться что это за домен. Если вбить почтовый ящик который не в базе с доменами, то поля все остальные пустые, только когда добавиться домен с таким почтовым, должна будет обновится информация по нему и появится связь почтового с этим доменом. Если по емейлу этому зарегистрировано несколько доменов, то должны показаться в таблице все домены которые связаны с этим почтовым.

3) Выносим в конфиг для админа редактирование текста отписки.
4) Страница отписки на ней указываются забитые причины отписки в административной части CRM.
В конфиге CRM нужен интерфейс через который можно будет создавать этот опросник, с возможностью включения и отключения поля с указанием своей причины.

Комит e08e6c2 - MBC-26 To Do (Отписка от рассылки и черный список емейлов)

  • Опросник (заполнение возможных причин) находится в Настройках - http://admincrm.mixtraff.com/?r=question/index
  • Текст отписки - в параметре "email_mass_unsub_text" Конфигурации (http://admincrm.mixtraff.com/?r=config/index)
  • вкл/откл поля "Укажите свою причину" на странице добавления емайла в "отписавшийся" - параметр "email_mass_custom_reason" Конфигурации
  • На данный момент на каждый единичный емайл формируется по 1 линку отписки, те если на домен отослано письмо с заголовком "To: email1, email2, email3" - то будет сформировано 3 линка в футере письма
  • Домены, емайлы которых занесены в "отписавшиеся", могут по прежнему находиться фильтром на стр. "Контакт - массовая рассылка", однако такие емайлы будут удалены из списка адресатов и в лог рассылки будет занесена соотв. запись.
  • Пример URL линка отписки - http://crm.mixtraff.com/?r=email/unsubscribe&email=email@yahoo.com&hash=e9df2f1d7ce6d87517e62db1b0c08457
